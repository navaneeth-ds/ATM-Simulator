# ATM Simulator (Python)

## About

This is my second Python mini project.

The project simulates a simple ATM system where users can create an account, check their balance, deposit money, withdraw money, change their PIN, and delete their account.

## Features

- Create Account
- Check Balance
- Deposit Money
- Withdraw Money
- Change PIN
- Delete Account
- Exit Program

## Concepts Used

- Python Dictionaries
- Nested Dictionaries
- Variables
- User Input
- if-elif-else
- while Loop
- CRUD Operations
